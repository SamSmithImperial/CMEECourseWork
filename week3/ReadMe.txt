Week 3 was an R bootcamp. The directories inside week 3 include 'code', 'results', 'images', 'sandbox' & 'data'.

Practical 1 (TreeHeight.R) was an exercise to calculate tree heights using triganometry with angle and distance from tree.
Practical 2 (Vectorize2.R) aimed to develop skills in enhancing functioning code in the interest of speed of computation using the systemtime function.
Practical 3 (Florida.R) - Is Florida getting warmer? This was my first time exposed to permuation analysis which calculated a p value for whether or not there is a correlation between time and temperature in Florida.
Practical 4 (PP_Dists.R) This pracical helped develop skills using tidyverse to wrangle data sets.
Practical 5 (PP_Regress.R) Firstly the task was to generate using ggplot a replica plot to the one on TheMulQuaBio. Secondly we had to make a dataframe with the mean log values for each feeding type.
Practical 6 (GPDD_Data.R) This was a more simple map exercise where long/lat points where plotted on a world map (i used ggplot for this but other functions are available.)
