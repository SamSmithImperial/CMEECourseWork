 
Week2 was a python bootcamp. The directories inside week2 include 'code', 'data', 'results', 'sandbox'

The week2 practicals include: 
    lc1.py
    lc2.py
    tuple.py
    dictionary.py
    align_seqs.py
    oaks_debugme.py

lc1, lc2, tuple, and dictionary practicals were tasks to implement new skills in writing for loops, list comprehensions, and understanding lists, tuples and dictionaries.

align_seqs and oaks_debugme were more challenging practicals to get to grips with working with DNA sequences in python and debugging python code respectively.
